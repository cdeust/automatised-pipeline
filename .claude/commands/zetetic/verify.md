# Quick Verify

Lightweight claim verification. Takes a single claim, does a fast source check.

## Instructions

1. Take the claim from $ARGUMENTS. If vague, sharpen it into a falsifiable proposition.

2. Search for 2 independent sources (papers, specs, official docs — not blog posts).

3. For each source found: state the exact finding and whether conditions match.

4. Verdict: Confirmed / Contested / Unsupported / Unknown.

5. If the claim needs deeper verification, suggest running the full `/skill-run verify-claim`.

Keep this fast — under 2 minutes. Depth is for the full skill; speed is for this command.
